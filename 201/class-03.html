HTML Lists

Ordered Lists

Odered lists are ordered 1. 2. 3. etc. An ordered list is is created with the <ol></ol> tag. The list items are each wrapped in a <li></li> tag.The ordered lists are great for step by step instructions such as recipes. For example:

        
            <ol>
                <li>item number one</li>
                <li>item number two</li>
                <li>item number three</li>
            </ol>

Unordered Lists

The unordered lists are the same as ordered ones just not in order or step by step. For example:

<ul> <li>list item random</li> <li>list item random</li> <li>list item random</li> </ul>

Definition Lists

The definition list is created with the <dl></dl> tag. These lists usually consists of a series of terms and their defintions. the

tag is for the definition terms and the
is for the defintion itself. For example:

<dl> <dt>Green Pesto</dt> <dd>A green basil pasta sauce</dd> <dt>Sundried Tomato Pesto</dt> <dd>A red pasta sauce mainly made of sundried tomatoes.</dd> </dl>
