CSS (Cascading Style Sheet)

What is CSS? CSS is a language for specifying how documents are presented to users. It is a rule based language that allows you to change the style and present your markup using whatever design you like. CSS includes:

  • Colours
  • Fonts
  • Sizes
  • Image properties
  • Box model (padding, border, margin)
  • Animations
  • Hover styles, focus styles (sudo classes)
  • Separation of concerns
